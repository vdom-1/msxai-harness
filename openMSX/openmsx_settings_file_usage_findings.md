# Settings

| Check | Setting | Description | Notes |
|:-----:|---------|-----------------------|-------|
| [  ] | `audio-inputfilename` | filename of the file where the sampler reads data from | |
| [  ] | `auto_save_replay_filename` | Filename of the replay file that will be saved when  auto_save_replay is enabled | |
| [  ] | `gui_font_default_filename` | TTF font filename for the default GUI font | |
| [  ] | `gui_font_mono_filename` | TTF font filename for the monospaced GUI font | |
| [  ] | `midi-out-logfilename` | filename of the file where the MIDI output is logged to | |
| [  ] | `printerlogfilename` | filename of the file where the printer output is logged to | |
| [  ] | `rs232-inputfilename` | filename of the file where the RS232 input is read from | |
| [  ] | `rs232-outputfilename` | filename of the file where the RS232 output is written to | |
